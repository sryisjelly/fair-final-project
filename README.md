# FaIR Final Project

Code and figures for my final project using FaIR to explore future global warming under SSP scenarios, including scenario divergence, a reduced CO2 emissions experiment, and comparison with IPCC AR6.

## Main notebook
- notebooks/cmip6_ssp_emissions_run.ipynb

## Experiments
1. Reproducing scenario divergence
2. Reduced fossil-fuel CO2 emissions in SSP2-4.5
3. Comparison with IPCC AR6

## Environment
Run:
- conda env create -f environment.yml
- conda activate fair

## Notes
The notebook uses local input files stored in the data/ folder.
